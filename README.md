# Cypress Demo

A project for exploring testing tools as part of a software engineering course.

## Overview

This repository contains examples and demonstrations of Cypress, an end-to-end testing framework for web applications.

## Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

## Installation

```bash
npm install
```

## Running Tests

```bash
npm test
```

## Project Structure

- `/cypress` - Contains test files and Cypress configuration
- `/cypress/e2e` - End-to-end test specifications
- `/cypress/fixtures` - Test data files
- `/cypress/support` - Custom commands and support files

## Purpose

This project is used for learning and practicing automated testing techniques using Cypress in a software engineering course.
